Global Warming Pong

Desktop

This was a project that I created in response to a brief I was given. The brief is as follows: 'Write a program that demonstrates the link between global warming and man made gas emissions. It could be an education tool, a game or a simulation?'

view repo on GitHub ►

visit website ►

Skills and Technologies Used

HTML, CSS & JavaScript

Aims

I decided to create a game for this, as the target audience were children/teens. I thought that this would make it more engaging and interesting compared to an educational tool. The game initially explains the effects of man made emissions on global warming and then leaves the player to try and stop Earth from overheating.

Successes

I created the design and assets for this game, which I think created a really cohesive look. I came up against a few challenges while calculating the associated rotation of the Earth according to mouse movement, but fortunately I was able to solve these and create a smooth experience for the player.

What Improvements Could I Make?

I would like to be able to make this available for use on mobile devices. I was unable to do this at the time as I was on a tight schedule, and my project was being judged on a laptop, so I optimised it to the target device.
